Java Web Viewer

Portable Java WebView app for local HTML files and web pages via http / https with useful window customization 

To change this page, simply replace the index.htm file in this app's directory with your html content, or point it to your web server to load a WebApp.
Example WebApp you could point to:   https://maps.google.com
java --module-path lib --add-modules javafx.controls,javafx.web -jar viewer.jar "https://maps.google.com" "G-Maps"
